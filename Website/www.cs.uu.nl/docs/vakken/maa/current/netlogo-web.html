Netlogo 6 web demos

This page contains links to Netlogo 6 files that can be run directly through the "netlogo-web" server, a version of NetLogo that runs entirely in the browser. The interface is not as pretty as in Netlogo itself, and some of demos simply don't run because Netlogo's web version unfortunately doesn't understand all Netlogo primitives, particular those that have to do with display and timing. Or the Netlogo program itself uses external libraries and then you're stuck as well. If you are still not discouraged then read on. Netlogo's web version is slower than the Netlogo interpreter itself, because of the implementation and because in newer versions of Netlogo the frame rate is clamped at 30fps. You can increase the frame rate by moving the upper slider to the right. Still, programs in Netlogo's web version often run shower than in Netlogo itself.
